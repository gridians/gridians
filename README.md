# 서비스 목적

- [x] 한 곳에서 모든 개발 관련 정보들을 관리할 수 있게 도모 **(프로젝트, 포트폴리오, 블로그 관리)**
- [x] 댓글 시스템으로 개발자간 소통 문제 해결 **(커뮤니케이션)**
- [x] 등록만 해도 언제든 헤드헌터들의 제안을 받을 수 있는 시스템 **(취업, 이직)**
- [x] 내가 좋아하는 개발자들을 즐겨찾기에 추가하여 꾸준히 팔로잉 **(소셜 네트워킹)**
